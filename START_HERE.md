## STEP 1: Copy SQL to Supabase (2 min)

```
1. Open: https://supabase.com → Your Project → SQL Editor → New Query
2. Open file: RUN_THIS_IN_SUPABASE.sql (in this project)
3. Copy EVERYTHING in that file
4. Paste into Supabase SQL Editor
5. Click RUN
6. Verify: Go to Table Editor → Should see 4 tables
```

**Tables you should see:**
- ✅ profiles
- ✅ projects  
- ✅ project_members
- ✅ tasks

---

## STEP 2: Deploy Server (30 sec)

```
1. Make Settings (click gear icon)
2. Find "Supabase" section
3. Click "Deploy Edge Function"
4. Wait for "Success" message
```

---

## 🎉 DONE! Now Use The App

### First Actions:
1. **Sign up** (full name, email, password)
2. **Create project** (Projects → + New Project)
3. **Add task** (click project → + Add Task)
4. **Drag task** between columns (To Do → In Progress → Done)

### Invite Team:
1. Ask them to sign up first
2. Go to project → Members → Invite Member
3. Enter their email → Select role → Invite
